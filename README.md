# TubeRipper-CLI
Small terminal app to download YouTube videos and playlists at MP3s.

Built using Python + Pytube.
## Demo

    1. Run the .exe file
    2. Paste in any YouTube video/playlist link
    3. File will save to your desktop
    4. You're done!
